# Visual Forge

Visual art methodology for AI-generated 2D game art. Teaches Claude composition, anatomy, color theory, perspective, lighting, and style — built on techniques from 50+ masters across painting, illustration, animation, and game art.

## Installation

```bash
claude plugins marketplace add wion4/claude-marketplace
claude plugins install visual-forge@wion4-marketplace
```

## Features

- **Anatomy & Proportion** — human proportion systems, contrapposto, gesture, foreshortening, face structure
- **Composition** — golden ratio, golden spiral, rule of thirds, leading lines, framing, negative space
- **Color Theory** — color wheel, temperature, value, saturation, palette construction, color psychology
- **Light & Shadow** — lighting setups (Rembrandt, chiaroscuro, rim), cast shadows, reflected light
- **Perspective** — 1/2/3 point, atmospheric perspective, depth cues
- **Silhouette & Shape Language** — readability, shape-personality mapping, design hierarchy
- **Style & Art Direction** — style spectrum, visual consistency, cultural visual DNA

## Commands

```
/visual-forge:draw create   — create concept art
/visual-forge:draw style    — build visual style guide
/visual-forge:draw review   — check anatomy/proportions
```
